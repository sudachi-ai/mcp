# Sudachi MCP

A Model Context Protocol (MCP) server for generating cross-chain transaction manifests. Enables AI agents to execute native L1 swaps (BTC, ETH, LTC, etc.) without bridges or wrapped tokens.

**Features:**
- Native cross-chain swaps
- Unsigned transaction generation for BTC, ETH, LTC, BCH, DOGE
- Real-time quote fetching
- Non-custodial - never touches private keys
